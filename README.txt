This is a simple Wordpress plugin that allows you to automatically display your installed plugins.

== Description ==

Pimp My Wordpress plugin automatically displays a list of your active plugins.  Not a big one - just a bit of fun.

== Installation ==

1. Upload `pimp-my-wordpress.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[pimpmywordpress]` shortcode to your body text any where you want the list of plugins displayed

== Frequently Asked Questions ==

= Can I change the seperator? =

Yes - just add the following argument to the shortcode 
[pimpmywordpress seperator=' / ']
